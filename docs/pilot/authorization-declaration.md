# Scan Authorization Declaration

By starting a scan, you confirm that:

1. You own or have explicit written authorization to test the target domain.
2. You understand SiberCheck will send HTTP requests to the target and linked assets.
3. You accept responsibility for ensuring the target is not a third-party system without permission.
4. You will not use SiberCheck to attack systems outside your declared scope.

Active scans (Deep/Code profiles) may send additional probes. Use staging targets when possible.

This declaration is recorded in the audit log with user ID, IP, timestamp, and target URL.
